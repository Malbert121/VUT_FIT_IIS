Step 1: Download the Project

  1. Download and open project folder.

Step 2: Set the Database Connection String

  1. Navigate to the file located at: Conventus.DAL\EfStructures\ConventusDbContextFactory.cs.
  2. Locate the line var connectionString and set it to the correct connection string for your database.

Step 3: Configure appsettings.json

  1. Open the Conventus.API folder.
  2. Ensure the following sections in appsettings.json are correctly configured:
    • Database Connection: Update the ConnectionStrings section with the database connection string.

Step 4: Open the Solution File

  1. Open the project in your IDE by double-clicking on the Conventus.sln file.

Step 5: Prepare the Database

  1. Apply any necessary database migrations to set up the database(console must be opened in Conventus.DAL): 
    
dotnet ef migrations add InitialCreate
dotnet ef database update
  2. Ensure the database is running and accessible with the configured connection string.

Step 6: Start the API Server

  1. Build and run the project(Conventus.API):
    • In Visual Studio, press F5 to start the API server.
    • If using a different IDE or terminal, navigate to the Conventus.API folder and run: 
      dotnet run
